Download all the required software:
	sudo apt-get install gcc automake autoconf libtool bison swig python-dev libpulse-dev

Get more required files from Github:
	git clone https://github.com/cmusphinx/sphinxbase.git

Go to the newly created "sphinxbase" directory:
	cd sphinxbase

Need to run this in "sphinxbase" directory (Generates the configure file):
	./autogen.sh

The "configure" file was generated, now run it:
	./configure

After getting some more files, run this:
	make clean all

Check everything was installed properly (*Don't have to do this):
	make check

Run another command:
	sudo make install

DONE INSTALLATION OF SPHINXBASE!


Export the library:
	export LD_LIBRARY_PATH=/usr/local/lib

Change some file:
	sudo nano /etc/ld.so.conf
	(add this to the file "/usr/local/lib") --second line after "include"

Export another library:
	export PKG_CONFIG_PATH=/usr/local/lib/pkgconfig


Download pocketsphinx:
	git clone https://github.com/cmusphinx/pocketsphinx.git

Go to the newly created "pocketsphinx" directory:
	cd pocketsphinx

Need to run this in "pocketsphinx" directory (Generates the configure file):
	./autogen.sh

The "configure" file was generated, now run it:
	./configure

After getting some more files, run this:
	make clean all

Run another command:
	sudo make install
